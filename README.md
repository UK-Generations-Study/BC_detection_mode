# BC_detection_mode
Analytical project using Generations and Screening data to explore differences between interval vs screen detected breast cancers.

## Project Background 
Initial work done by Louise. The analytical dataset was initially collated in Safe Haven, then transferred in the same format to RDS where analysis was carried out using Stata. To ensure reproducibility and consistency with current projects, this project is now to be translated into R using RDS datasets. The code will be stored in GitHub.
